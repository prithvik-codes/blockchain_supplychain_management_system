# Offline Blockchain Supply Chain Tracker (tracking + QR + CSV)

## What
Simple Flask app to:
- Add product events (product id, location, action)
- Track a product's journey (visual timeline)
- Generate QR code per product linking to its track page
- View blockchain, validate integrity, export CSV, tamper demo
